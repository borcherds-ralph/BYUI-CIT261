Local Storage API

Study Sources:
MDN web docs - Local Storage
W3Schools - Local Storage API

Scope:

HTML web storage allows us to store data within the user's browser; additionally, it is not sent to the web server.

HTML Web Storage Objects

  • window.localStorage : stores data with no expiration date
  • window.sessionStorage : stores data for one session (data is lost when the broser tab is closed)
